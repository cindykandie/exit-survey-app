# Exit Survey React Native App

![Exit Survey App Logo](./assets/images/welcome.png)

## Introduction

The Exit Survey React Native App is designed to collect valuable feedback from departing employees, helping organizations gather insights for improvement. This app is built using Expo and provides an easy-to-use interface for employees to share their feedback confidentially.

## Features

- **User-Friendly Interface**: A streamlined and intuitive design for ease of use.
- **Secure and Confidential**: All responses are handled confidentially to ensure employee privacy.
- **Survey Customization**: The app allows for customizable survey questions.
- **Data Collection**: Accurate and secure data collection from departing employees.
- **Feedback Submission**: Employees can submit feedback with the tap of a button.
- **Offline Capability**: Support for completing surveys offline and syncing data later.
- **Multilingual Support** (Optional): Choose your preferred survey language.
- **Contact Support**: Reach out to us if you have questions or need assistance.

## Getting Started

To run the Exit Survey React Native App on your local machine, follow these steps:

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/cindykandie/exit-survey-app.git
   ```

2. **Navigate to the Project Directory**:

   ```bash
   cd exit-survey-app
   ```

3. **Install Dependencies**:

   ```bash
   npm install
   ```

4. **Start the Expo Development Server**:

   ```bash
   npx expo start
   ```

5. **Open the App**:

   Use the Expo Go app on your iOS or Android device to scan the QR code displayed in the terminal or in the Expo DevTools that open in your web browser.
